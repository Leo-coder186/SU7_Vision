# CarX 看车项目

基于 Vue 3 + Three.js 的 3D 汽车展示应用，支持材质切换、开关门动画、车灯效果等交互功能。

---

## 技术栈

| 分类 | 技术 | 说明 |
|------|------|------|
| 框架 | Vue 3 | 组合式 API（`<script setup>`） |
| 构建 | Vite 5 | 快速的开发服务器和构建工具 |
| 3D渲染 | TresJS + Three.js | Vue 3 的 Three.js 封装库 |
| 动画 | Tween.js | 补间动画（颜色、位置、缩放过渡） |
| GUI | Lil-GUI / Leches | 运行时参数调节面板 |
| 路由 | Vue Router | 页面路由管理 |
| 状态 | mitt（事件总线） | 跨组件通信 |

## 核心模块说明

### 1. CarModel.vue（核心枢纽）

负责协调所有子模块，是项目的控制中心。

```
CarModel.vue
    │
    ├── useGLTF() 加载模型
    │       │
    │       ▼
    │   useModelInit() 初始化
    │       │
    │       ├── 提取 sprites（车门粒子）
    │       ├── 提取 lensflares（车灯位置）
    │       └── 提取 marker（尺寸标注）
    │
    ├── useMeshStatic() 渲染材质球
    │       │
    │       └── 创建独立场景渲染材质选择球
    │
    ├── useAnimation() 动画控制
    │       │
    │       ├── 渲染车门粒子
    │       ├── 添加点击事件（开关门）
    │       └── 处理轮胎旋转
    │
    └── eventBus 监听
            │
            ├── changeCarLight → 创建/移除车灯光源
            ├── changeCarStart → 切换地面 / 轮胎旋转
            └── changeCarSize → 显示/隐藏尺寸标注
```

### 3. 交互数据流

```
用户操作                    事件传递                   响应结果
─────────────────────────────────────────────────────────────────

点击"开启车灯"
    │
    ▼
ViewController ──── eventBus ────► CarModel
                                        │
                                        ├──► useLensflare()
                                        │         │
                                        │         ▼
                                        │    创建 PointLight + Lensflare
                                        │         │
                                        └──► scene.add(light)
                                                  │
                                                  ▼
                                            车灯亮起

点击材质球
    │
    ▼
useMeshStatic ──── pick() ────► 射线检测
                                     │
                                     ▼
                              获取 uniqId
                                     │
                                     ▼
                              getMaterialPlanById()
                                     │
                                     ▼
                              应用材质到汽车模型
                                     │
                                     ├──► 颜色渐变动画
                                     └──► 参数直接赋值

鼠标移动
    │
    ▼
useMeshStatic ──── mousemove ────► pick()
                                          │
                                          ▼
                                    悬停检测
                                          │
                                     ┌────┴────┐
                                     │         │
                                     ▼         ▼
                                 悬停        离开
                                   │         │
                                   ▼         ▼
                              放大材质球    恢复大小
                              旋转材质球
```
